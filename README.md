# newsreader
An RSS feed based news app. (Android)

The news provided consists of following categories:

1. Highlights.
2. Worldwide.
3. Nation. (Depends on edition)
4. Business.
5. Technology. 
6. Sports.
7. Entertainment.
8. Science.
9. Health.


This app makes use of RSS feed provided by Google News.
